Política de Privacidade

Última atualização: 06 de junho de 2025

Esta política de privacidade descreve como o jogo QUICKVENTURE (doravante "o jogo") lida com informações dos usuários.

Em caso de dúvidas, entre em contato: bereqdev@gmail.com

1. Dados coletados automaticamente

O jogo utiliza o serviço Photon Engine (da Exit Games GmbH) para possibilitar partidas online e funcionalidade multiplayer.

Ao utilizar o jogo, o Photon pode coletar automaticamente dados técnicos como:

  • Endereço IP
  • Informações sobre o dispositivo (modelo, sistema operacional, identificador de dispositivo)
  • Estatísticas de conexão (latência, ping, perda de pacotes)
  • Dados de eventos relacionados ao funcionamento da rede multiplayer

Esses dados são coletados e processados de acordo com a política de privacidade da Exit Games GmbH.

2. Dados pessoais

O jogo em si não coleta, armazena nem compartilha dados pessoais diretamente, como email, número de telefone ou localização precisa.

As únicas informações transmitidas são aquelas necessárias para viabilizar o funcionamento da partida online.

3. Dados de terceiros

Se no futuro o jogo integrar serviços de terceiros (ex: anúncios, análise de uso, serviços de autenticação), os usuários serão informados por meio de atualização nesta política.

Atualmente, nenhum outro serviço de terceiros além do Photon é utilizado.

4. Alterações nesta política

Podemos atualizar esta política de tempos em tempos. A data de "última atualização" no topo da página indica a versão mais recente.

Recomendamos que você revise esta página periodicamente para se manter informado sobre eventuais alterações.
